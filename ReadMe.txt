# Analyse de la Qualité du Vin

Application Streamlit pour l'analyse exploratoire et la modélisation de la qualité des vins portugais Vinho Verde.

## Lien Git

Repository : https://github.com/jeremyconti613/wine-quality.git

## Installation

```bash
pip install -r requirements.txt
```

## Lancer l'application Streamlit

```bash
streamlit run streamlit_app.py
```

L'application sera accessible à l'adresse : http://localhost:8501

## Contenu

L'application propose plusieurs sections :
- Compréhension des données (contexte, structure, variables)
- Préparation des données
- Vérification de la qualité des données
- Visualisations exploratoires
- Relations entre variables et hypothèses
- Régression linéaire (prédiction de la note exacte)
- Classification avec régression logistique
- Interprétation et limites
